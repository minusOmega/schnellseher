# Schnellseher

This project uses React + TypeScript with Vite.

## Requirements

- Node.js 22 (`.nvmrc` is set to `22`)
- npm

## Scripts

- `npm run dev` or `npm start`: start Vite dev server
- `npm run build`: production build to `build/`
- `npm run preview`: preview the production build locally
- `npm test`: run Vitest in watch mode
- `npm run test:run`: run Vitest once (CI mode)

## Static assets

- Public static files are served from `public/`.
- App entry HTML is `index.html` at the repository root.
